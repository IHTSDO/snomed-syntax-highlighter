SNOMED CT Expression Formatter with Monaco Editor

This webapp uses Monaco Editor for robust syntax highlighting and formatting of SNOMED CT expressions. Use the links below to test with different expressions:

Features:

  • Concept IDs (9-digit numbers) - Highlighted in grey
  • Pipes (|) - Highlighted in blue
  • Keywords (id) - Highlighted in red
  • Operators (:, =, <, <<, >>) - Highlighted in red
  • Brackets ({}, [], ()) - Highlighted in red
  • Role Groups (@Finding, @Relationship) - Highlighted in red
  • Separators (commas) - Highlighted in red
  • Normalization - Cleans up spacing around descriptions
  • Indentation - Formats attribute groups with proper indentation
  • Nested Expressions - Handles complex nested structures

Usage:

To use this formatter in your application, embed it with a URL like:

index.html?expression=YOUR_SNOMED_EXPRESSION_HERE

The expression parameter should be URL-encoded if it contains special characters.
