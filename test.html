SNOMED CT Expression Formatter with Prism.js

This webapp uses Prism.js for robust syntax highlighting of SNOMED CT expressions. Use the links below to test with different expressions:

Features:

  • Concept IDs (9-digit numbers) - Highlighted in blue
  • Descriptions (text between | bars) - Highlighted in black
  • Operators (:, =, <, <<, >>) - Highlighted in red
  • Brackets ({}, [], ()) - Highlighted in gray
  • Keywords (+id) - Highlighted in green
  • Role Groups (@Finding, @Relationship) - Highlighted in purple
  • Separators (commas) - Highlighted in gray
  • Normalization - Cleans up spacing around descriptions
  • Indentation - Formats attribute groups with proper indentation
  • Nested Expressions - Handles complex nested structures

Usage:

To use this formatter in your application, embed it with a URL like:

index.html?expression=YOUR_SNOMED_EXPRESSION_HERE

The expression parameter should be URL-encoded if it contains special characters.
